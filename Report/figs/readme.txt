how to write farsi in ipe?
	go to edit/document properties -> latex engine: xetex and pase this code on the latex preamble box:
	\ipedefinecolors{}
	\usepackage[RTLdocument=off]{xepersian}
	\settextfont{Yas}
	\setdigitfont{Yas}
